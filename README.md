A multi-agent AI research assistant built using LangChain, Ollama, FastAPI, and Streamlit. supports semantic document retrieval, question-answering, summarization, and comparison using local LLMs.

Features:
- Multi-agent AI architecture
- Retrieval-Augmented Generation (RAG)
- Semantic document search
- Local LLM integration using Ollama
- FastAPI backend
- Streamlit frontend
- PDF document querying
- Summarization and comparison agents

Tech Stack:
-Python
- LangChain
- Ollama
- FastAPI
- Streamlit

Frontend → FastAPI Backend →Agent Router →RAG Pipeline →Local LLM
Sentence Transformers
PyMuPDF
